## 인공지능전공 전공 스터디 (동계방학)

Cursor 기반 바이브 코딩(Vibe Coding) 방식으로  
다양한 AI 응용 프로그램 30종을 직접 구현하며 학습했습니다.

LLM 활용, 프롬프트 엔지니어링, 간단한 자동화/생성형 AI 서비스 구현 경험을 쌓았고,  
학습 과정에서 제작한 프로젝트들을 본 저장소에 정리했습니다.

### 주요 학습 내용
- Cursor를 활용한 AI 기반 개발 워크플로우 경험
- 생성형 AI API 활용 및 서비스 구현
- 빠른 프로토타이핑 및 기능 검증 경험
- 아이디어를 실제 동작하는 프로그램으로 구현하는 개발 역량 강화
