### 제목
MCP 이해하기: 클로드와의 연계 및 설치 방법

### 주요 내용
- MCP는 클로드와 같은 LLM 기반 서비스에 추가 기능을 제공하는 모듈로, 브라우저의 확장 프로그램에 비유할 수 있다.
- 예를 들어, 'Desktop Commander' MCP를 클로드에 추가하면 운영체제 명령어를 수행할 수 있다.
- MCP는 서로 협업이 가능하여 다양한 기능을 조합할 수 있다.
- MCP는 클로드뿐만 아니라 AI 코드 작성 도구인 '커서'에도 적용할 수 있다.

### 결론/요점
MCP는 클로드와 커서에 추가 기능을 제공하여 사용자가 더 효율적으로 작업을 수행할 수 있도록 돕는다. MCP 설치를 위해서는 스미더리 API 설정, Node.js 설치, 설치 명령어 입력, 클로드 개발자 모드 활성화가 필요하다.
