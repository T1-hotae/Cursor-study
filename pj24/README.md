# 블로그 글 생성기

키워드, 나의 관점(200자 내외), 참고 자료(텍스트 파일)를 입력하면 GPT API로 창의적인 블로그 글을 생성하는 프로그램입니다.

## 사용 모델: gpt-4o-mini

- **가성비**: 입력/출력 단가가 GPT-4 대비 훨씬 낮고, 블로그 글 수준의 창의적 글쓰기에 충분한 품질을 제공합니다.
- **컨텍스트**: 128K 토큰 지원.
- [OpenAI 가격 정보](https://platform.openai.com/docs/pricing)에서 최신 요금을 확인할 수 있습니다.

## 설치

1. Python 3.9 이상이 필요합니다.

2. 의존성 설치:

   ```bash
   pip install -r requirements.txt
   ```

3. API 키 설정:
   - [OpenAI API 키](https://platform.openai.com/api-keys)에서 키를 발급받습니다.
   - 프로젝트 폴더에 `.env` 파일을 만들고 다음 내용을 넣습니다 (실제 키로 교체).

   ```
   OPENAI_API_KEY=sk-your-actual-api-key
   ```

   `.env.example` 파일을 복사해 `.env`로 이름을 바꾼 뒤 키만 수정해도 됩니다.

## 실행

### GUI로 실행 (파일 첨부)

```bash
python blog_generator_gui.py
```

- **키워드** 입력란에 주제를 입력하고, **나의 관점**란에 본인의 시각을 200자 내외로 쓸 수 있습니다(선택).
- **파일 첨부** 버튼으로 참고 자료 텍스트 파일을 선택할 수 있습니다.
- **블로그 글 생성** 버튼을 누르면 생성이 진행되고, 결과가 아래 영역에 표시됩니다.
- 나의 관점·참고 자료는 없어도 되며, 키워드만으로도 생성할 수 있습니다.

### 터미널(CLI)로 실행

**방법 1: 키워드 + 나의 관점 + 참고 자료 파일**

```bash
python blog_generator.py 생산성 -p "생산성은 효율이 아니라 우선순위에서 나온다고 생각합니다." -f references.txt
```

**방법 2: 키워드만 인자로 전달**

```bash
python blog_generator.py 생산성
```

**방법 3: 실행 후 키워드·파일 입력**

```bash
python blog_generator.py
```

실행 후 키워드와 참고 자료 텍스트 파일 경로를 입력할 수 있습니다. 참고 자료는 없으면 Enter로 건너뛰면 됩니다.

- **나의 관점**: 200자 내외로 작성자의 시각을 적으면, 그 톤과 관점을 반영해 블로그 글이 작성됩니다. `-p "..."` 또는 실행 후 프롬프트에서 입력할 수 있습니다.
- **참고 자료**: UTF-8 텍스트 파일(.txt 등)을 지정하면, 그 내용을 반영해 블로그 글이 작성됩니다.

## 출력

키워드를 바탕으로 한 창의적인 블로그 글이 한국어로 출력됩니다.  
출력된 글을 그대로 블로그에 붙여넣어 사용할 수 있습니다.
